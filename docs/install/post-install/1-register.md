# Register a Sysadmin account

The first account created will be Sysadmin and allow you to continue configuring your instance.

Point your browser to `https://<your-elabftw-site.org>/register.php` and create a local account.

![elabftw register](/img/install-register.webp)

# Go to Sysadmin panel

After creating your account and logging in, head to the Sysadmin panel:

![sysadmin menu](/img/install-sysadmin-panel-menu.webp)
